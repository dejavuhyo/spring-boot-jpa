# Spring Boot JPA

## 1. 설명
Spring Boot JPA 예제이다. 포트는 8080을 사용한다.

## 2. 개발환경

* OpenJDK 17

* spring-boot 3.0.1

## 3. API 실행

### 1) 전체 조회

* GET
  - htttp://localhost:8080/spring-boot-jpa/member

### 2) ID 조회

* GET
  - htttp://localhost:8080/spring-boot-jpa/member/1

### 3) 저장

* POST
  - htttp://localhost:8080/spring-boot-jpa/member

### 4) 수정

* PUT
  - htttp://localhost:8080/spring-boot-jpa/member/1

### 5) 삭제

* DELETE
  - htttp://localhost:8080/spring-boot-jpa/member/1
